This repository contains Java practical programs for the subject **Object Oriented Programming**
 (BE04000231) under the GTU BE Semester 4 curriculum.

It covers a wide range of topics from basic programming to advanced concepts such as object oriented programming, inheritance, polymorphism, exception handling, multithreading, file handling, collections, and JavaFX.

The programs are organized in a clear and structured manner, where each program focuses on a specific concept. All code is written in a simple and clean format, making it easy to understand, execute, and useful for learning, practice, and exam preparation.

**Topics Covered:**

Basic Java Programming – programs based on input, calculations, unit conversion, and logical problem solving.

Object Oriented Programming (Classes and Objects) – creation and use of classes, objects, and methods.

Constructors and Copy Constructor – default, parameterized, and copy constructors for object initialization.

Static Members and Inner Classes – use of static variables/methods and implementation of inner classes.

Method Overloading – defining multiple methods with same name but different parameters.

Inheritance – creating base and derived classes to reuse code.

Polymorphism – method overriding and runtime behavior of objects.

Interfaces – implementing multiple interfaces and defining abstract methods.

Abstract Classes – use of abstract methods and subclass implementation.

Packages – organizing classes into different packages for better structure.

Exception Handling – handling runtime errors using try-catch-finally and custom exceptions.

Multithreading – creating and running multiple threads.

Synchronization – controlling thread execution to avoid conflicts.

File Handling – reading and writing data using FileReader and BufferedReader.

Collection Framework – using ArrayList, HashMap, and LinkedList for data storage and processing.

Generics – writing flexible and reusable code using generic methods.

JavaFX (GUI Development) – creating user interfaces with forms, text, and graphical components.
